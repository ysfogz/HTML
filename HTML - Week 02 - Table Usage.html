Merhaba arkadaslar,

Ilisikte resmi verilen layoutu, tablo, img ve diger ilgili tagleri kullanarak gerceklestiriniz.

Kriterler (Karsilanmadigi durumda kesilecek puan miktarini kriterlerin yaninda gorebilirsiniz)
- Programin istenilen sekilde programlanmasi (-40p)
- W3C Validatori ile sayfanizi kontrol ediniz ve SCREENSHOT'ini aliniz (https://validator.w3.org/#validate_by_input+with_options). (-10p)
- Lütfen html'de hizalama (indentation) azami dikkat ediniz.(-15p)
- Hiçbir şekilde: CSS veya JavaScript kullanılmayacak. Burdaki amacımız en sade şekilde HTML Tag'leriyle uğraşmak. (-10p)
